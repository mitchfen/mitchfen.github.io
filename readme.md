## How to serve the site locally

```bash
# Using Docker
sudo docker run --rm -p 80:80 ghcr.io/mitchfen/mitchfen.github.io:latest
```

```bash
# Using npm
git clone https://github.com/mitchfen/mitchfen.github.io.git
cd mitchfen.github.io
npm ci
npm start
```
